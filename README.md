## 💳 Digital Wallet Payment System

A secure wallet platform built with *Java OOP concepts*:
- Encapsulation for data security  
- Static variables for global cashback policy  
- Exception handling for safe transactions  

---

## 🚀 Features
- Secure balance handling  
- Wallet-to-wallet transfers  
- Global cashback rewards  
- Exception-safe transaction logic
